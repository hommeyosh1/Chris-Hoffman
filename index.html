Featured Game: Circle Drop

Explore

Circle Drop is a browser-based puzzle game I developed using HTML, CSS, and JavaScript, with the Canvas API handling the dynamic visuals and animations. The game is inspired by Tetris but uses patterned circles instead of blocks, challenging players to match each falling circle with the correct platform. I designed the entire gameplay loop from scratch, including collision detection, increasing difficulty, and responsive controls for both desktop and mobile touchscreen play. I used HTML, CSS, and JavaScript to design the game to play seamlessly on both browsers and mobile devices, ensuring smooth touch controls for an enhanced mobile experience. I handled every aspect of the project independently—from game design and programming to testing and deployment. The game is hosted on Replit, allowing for easy access and sharing. This project demonstrates my ability to bring a full concept to life using web technologies and showcases my skills in logic design, user interaction, and browser-based game development.

Open World FPS TechDemo in Unity: A First-Person Shooter Experience

I created a Unity tech demo featuring an open world with green grass and animated trees. The demo utilizes Unity’s terrain feature, where I painted textures to give the environment a natural look. It includes a first-person character equipped with an animated gun, which fires a shot with a sound effect when the right mouse button is pressed. The player can move using the WASD keys, aim with the mouse, jump, and the crosshairs are implemented using Unity's UI system. Additionally, I added background sound using an MP3 file to enhance the atmosphere. This project demonstrates my skills in Unity, animation, sound integration, and first-person controls, all programmed in C# using Visual Studio Code (VSC).
